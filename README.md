توضیحات:

پروژه ی گام یار برای افرادی است که از کلاس نهم خود را تمام کرده اند و می خواهند
انتخاب رشته کنند
این برنامه تا حد زیادی برای افراد راهنمایی های خوبی می کند.

این برنامه برای سیستم عامل های لینوکس و ویندوز و مک کارایی دارد و شما با توجه 
به سیستم عامل خود فایل داخل پوشه run را اجرا کنید.

روش اجرا:

وارد پوشه پروژه شوید
فایل های داخل پوشه run را اجرا کنید.
و سپس 
api GapGPT را وارد کنید.

و سپس وارد آدرس خروجی بشوید و با توجه به علایق و توضیحات 
وارد کنید و جواب خود را بگیرید.

روش اجرا به صورت کد برای سیستم عامل های مختلف:

اگر سیستم عامل شما ویندوز است فایل run.bat را اجرا کنید
اگر سیستم عامل شما لینوکس توزیع(ابونتو و دبیان است) فایل runFish.sh یا runBash.sh را اجرا کنید  
که آن هم به ترمینال شما بستگی دارد.
